# CareerWay Hub: Ab Manzil Dur Nahin

An AI-powered job portal built with **Python (Flask)** and **Google Gemini 1.5 Flash API**.

## 🚀 Key Features
* **Sarkari Scraper:** Real-time government job updates.
* **AI Auto-Write:** Automated cover letter generation.
* **Recruiter Dashboard:** Manage applications efficiently.

## 🛠️ Tech Stack
* **Backend:** Flask (Python)
* **Database:** SQLite
* **AI Engine:** Google Gemini API
* **Frontend:** Bootstrap 5 (Dark Mode supported)
